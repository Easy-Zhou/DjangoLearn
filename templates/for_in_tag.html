for in loop 没有 break和continue语句

正序遍历forloop

    {% for book in books %}
  • {{ book }}
    • {% endfor %}

倒序遍历的forloop

    {% for book in books reversed %} {# 将列表倒序遍历 #}
  1. {{ book }}
    2. {% endfor %}

遍历字典的key

    {% for key in persons.keys %}
  • {{ key }}
    • {% endfor %}

遍历字典的values

    {% for value in persons.values %}
  • {{ value }}
    • {% endfor %}

遍历字典的items

    {% for key,value in persons.items %}
  • {{ key }}:{{ value }}
    • {% endfor %}
